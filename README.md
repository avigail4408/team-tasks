# Team Tasks Management System

מערכת Full-Stack לניהול משימות צוותיות המבוססת על Django (Python) בצד השרת.

## טכנולוגיות מרכזיות:
- Python & Django
- SQLite (Development DB)
- REST APIs

## הרצה מקומית:
1. התקנת הדרישות: `pip install -r requirements.txt`
2. הרצת מיגרציות: `python manage.py migrate`
3. הרצת השרת: `python manage.py runserver`
